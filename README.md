# ai-sub-showcase

This repository showcases subtitles generated using [ai-sub](https://github.com/FlippFuzz/ai-sub). 

All subtitles featured here are for publicly available YouTube videos; this collection does not include content from members-only or private videos.

> [!WARNING]  
> **Accuracy Disclaimer:** These subtitles are generated entirely by AI and have **not been manually reviewed**. Because AI can make mistakes, the text may be inaccurate, contain transcription errors, or misrepresent the original audio. Please use them with this in mind.
